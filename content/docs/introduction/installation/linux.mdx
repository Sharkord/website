# Linux Installation

Download the latest Linux x64 binary and run it:

```bash
curl -L https://github.com/sharkord/sharkord/releases/latest/download/sharkord-linux-x64 -o sharkord
chmod +x sharkord
./sharkord
```

Then open `http://localhost:4991` in your browser.

If hosted remotely, use your server IP or domain instead of `localhost`.
