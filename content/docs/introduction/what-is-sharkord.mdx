# What is Sharkord?

Sharkord is a self-hosted communication platform designed to provide core Discord-like features on your own infrastructure. It shares the philosophy of Teamspeak but with a clean interface and a focus on ease of use.

You host voice channels, text chat, and file sharing while keeping full ownership of data and operational control.

## Core Principles

- Self-hosted
- Lightweight
- Clean interface
- No surveying or telemetry
- Open source
- No paywalls or premium features
- Easy to deploy
